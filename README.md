# asm_minilibc
 Some functions of the libc recoded in asm

######List of the functions:
```
     - index
     - memcpy
     - memmove
     - memset
     - rindex
     - strcasecmp
     - strchr
     - strcmp
     - strcspn
     - strlen
     - strncmp
     - strpbrk
     - strstr
```
